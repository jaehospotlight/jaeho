## How to write a new post

Create a new file in the `_posts` folder with this naming format:

`YYYY-MM-DD-your-post-title.md`

At the top of each post, include the front matter:

```
---
layout: post
title: "Your Post Title"
subtitle: "An optional subtitle"
date: 2026-03-07
---
```

Then write your content in Markdown below it. That's it — GitHub Pages will automatically build and publish your new post.

## Markdown basics

You can write in **bold**, *italics*, and include [links](https://example.com).

> Blockquotes look like this. Good for pulling out key ideas.

You can also include images, code blocks, and lists. Markdown makes it easy to focus on writing rather than formatting.
